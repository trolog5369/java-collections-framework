This section documents my hands-on understanding of **Java Arrays**, treated as a core building block for all future data structures.

### What I Practiced & Understood

- Fixed-size nature of arrays and its implications
- Zero-based indexing and bounds safety
- Internal memory behavior (why resizing is not allowed)
- Read vs write operations and their trade-offs
- Forward and reverse traversal techniques
- Enhanced for-loops vs traditional loops
- Using `java.util.Arrays` utility methods
- Stream-based traversal for cleaner iteration

### Key Files

- **`Array.java`**
  - Conceptual notes and limitations of arrays
  - Why arrays are fast for access but rigid for modification

- **`TheArrays.java`**
  - Array creation techniques
  - Index-based insertion and overwriting
  - Traversal (normal, reverse, enhanced for-loop)
  - Stream-based iteration
  - Common runtime pitfalls (index out of bounds)

- **`ArraysUtility.java`**
  - Practical use of `Arrays.toString()`

  - Reinforces clean inspection of array data
